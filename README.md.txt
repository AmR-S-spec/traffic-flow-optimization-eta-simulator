# Traffic Flow Optimization and Delivery ETA Simulator

## Overview

This project combines Graph Theory and Transportation Network Analysis to study traffic flow optimization and estimate delivery times on real-world road networks.

The project consists of:

1. Traffic Flow Optimization using Graph Theory
2. Delivery ETA Simulator built with Streamlit

Road network data is obtained from OpenStreetMap and analyzed using graph-based algorithms.

## Features

* Road network extraction from OpenStreetMap
* Graph-based route modelling
* Shortest path computation
* Traffic condition simulation
* Delivery ETA estimation
* Route visualization
* Interactive Streamlit interface

## Technologies Used
